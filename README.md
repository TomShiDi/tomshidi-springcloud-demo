# tomshidi-springcloud-demo
## Subject
Demo basing on SpringCloud for creating some useful components

## Features
+ [Dynamic log level for logback](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/core-demo/src/main/java/com/tomshidi/demo/log/DynamicLogLevelConfig.java) 
+ [External properties file loading](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/core-demo/src/main/java/com/tomshidi/demo/factory/TomPropertySourceFactory.java)
+ [Loading for Any yaml file](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/core-demo/src/main/java/com/tomshidi/demo/processor/ThirdPartYamlProcessor.java)
+ [Dynamic server name for rest-template and openfeign](https://github.com/TomShiDi/tomshidi-springcloud-demo/tree/master/core-demo/src/main/java/com/tomshidi/demo/interceptor)
+ [Http Client basing on feign](https://github.com/TomShiDi/tomshidi-springcloud-demo/tree/master/core-demo/src/main/java/com/tomshidi/demo/client)
+ [Refreshable config by using nacos and @Value](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/core-demo/src/main/java/com/tomshidi/demo/config/NacosRefreshableConfig.java)
+ [Example for @RestControllerAdvice](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/core-demo/src/main/java/com/tomshidi/demo/config/ControllerAdviceConfiguration.java)
+ [Handy json parsing tool](https://github.com/TomShiDi/tomshidi-springcloud-demo/blob/master/base/src/main/java/com/tomshidi/base/utils/JsonUtil.java)
+ [Closure-Table solution for Tree hierarchy](https://github.com/TomShiDi/tomshidi-springcloud-demo/tree/master/mybatis-demo/src/main/java/com/tomshidi/mybatis)
+ [Dynamic field crypt based on spring aop](https://github.com/TomShiDi/tomshidi-springcloud-demo/tree/master/base/src/main/java/com/tomshidi/base/encrypt)
+ [Dynamic field crypt based on mybatis and mybatis-plus](https://github.com/TomShiDi/tomshidi-springcloud-demo/tree/master/mybatis-demo)
